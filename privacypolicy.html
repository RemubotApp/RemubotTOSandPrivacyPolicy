Remubot Privacy Policy

Effective Date: 01/07/25

What Data We Collect

Remubot only collects log-related data from the Discord guild (server) where it is invited. This may include:

  • Message edits and deletions
  • User joins and leaves
  • Channel or role changes
  • Moderation actions (e.g., bans, kicks)

Remubot does not collect or store direct messages, personal information, or any data outside the server where it is active.

How Data Is Used

The collected data is used solely for generating logs visible within the server for moderation purposes. It is not shared or sold to any third parties.

Data Storage

Remubot does not permanently store any data unless persistent logging is enabled in a future update. By default, all processing is in-memory or temporary.

Security

Remubot follows standard best practices to ensure data security. We recommend limiting the bot's permissions as needed to reduce risk.

Your Control

  • Removing the bot from your server stops all data collection immediately.
  • If persistent storage is implemented, data deletion requests will be honored on request.
